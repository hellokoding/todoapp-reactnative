# Todo App with React Native

### Guide
https://hellokoding.com/todo-app-with-react-native/

### What you'll need
- NodeJS
- NPM
- React Native 0.28+

### Stack
- ES6
- React Native

### Run
```bash
git clone git@github.com:hellokoding/todoapp-reactnative.git
cd todoapp-reactnative
npm install
react-native link
react-native run-ios
```
